# CDER GraphRAG Chatbot

Welcome to the **CDER GraphRAG System** - A hybrid retrieval-augmented generation system for Parallel and Distributed Computing education.

## Features

This chatbot provides **4 different retrieval approaches** for each question:

1. **No-RAG (Baseline)** - Answer without external retrieval
2. **Vector-Only RAG** - Semantic similarity search using embeddings
3. **Graph-Only RAG** - Entity and relationship-based graph traversal
4. **Hybrid RAG** - Combined vector and graph retrieval

## How to Use

Simply ask any question about Parallel and Distributed Computing, and you'll receive 4 different answers with:
- Full answer text
- Citations
- Performance metrics (latency, tokens, confidence)
- Side-by-side comparison

## Example Questions

- "What is parallel computing?"
- "Explain MapReduce architecture"
- "What are the benefits of distributed systems?"
- "How does load balancing work?"
